{{ 'Bank reporting' | uppercase }} {{ 'Usage guide' | uppercase }}

The Bank reporting system allows for exporting partners, merchants and transactions, as well as searching transaction, and importing transactions from an XML file.


{{ 'Export' | uppercase }}

You can export partners, merchants, or transaction on the Home page by clicking the export buttons for each type.

You can also export transactions on the transaction page described below.

{{ 'Transactions' | uppercase }}

There are various options when it comes to the transactions.

Search transactions

You can search transactions by numerous transaction properties like beneficiary iban, amount details, create date, etc. You can also search all transactions by not specifying any criteria.

After searching you are displayed the search result in a sortable table with pagination.

You can reset the search by clicking on the reset button.

Partner and merchant filters

Partners can be filtered out by entering at least three symbols in the partner field. Then if matched partners are found, select one from the list. The list may contain only a few of the matched records, so use the export functionality to be more specific about the partner name.

Merchant field is initially disabled. It needs a partner to be selected before it enables. Searching this field is as is the partner search.

Export transactions

You can export transactions by clicking the export button on the search transactions screen. The export will take into consideration the current search criteria and export accordingly.

Import transactions

There is a functionality to import transaction from an XML file. Since the file itself does not contain information about the partner and the merchant it is thus required to fill in the partner and merchant information on the search screen.
